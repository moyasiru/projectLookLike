# 온라인홈쇼핑몰 LookLike


<h1>프로젝트 소개</h1>
  
<h3>저희 홈페이지 LookLike는 온라인쇼핑몰로 관리자가 상품등록을 하면 일반 유저들이 상품을 사는 일반적인 옷 쇼핑몰입니다.</h3>

수업내용에 주로 배운 Spring Boot, Maven, JSP 등을 팀원들의 합의 하에 진행 하였고, 추가적으로 오픈 api 를 이용하여 주소, 결제 쪽에 사용하여 구현해보았습니다.
